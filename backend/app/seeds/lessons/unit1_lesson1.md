# Rational vs. Irrational Numbers

## Learning Objective
Students will be able to distinguish between rational and irrational numbers, explain why certain numbers (like √2 and π) are irrational, and recognize that every real number has a decimal expansion that either terminates or repeats (rational) or goes on forever without repeating (irrational).

---

## Key Vocabulary

- **Rational number** — Any number that can be written as a fraction p/q where p and q are integers and q ≠ 0. The decimal expansion either terminates or repeats.
- **Irrational number** — A number that cannot be written as a fraction of two integers. Its decimal expansion never terminates and never repeats.
- **Integer** — The set of whole numbers and their negatives: ..., −3, −2, −1, 0, 1, 2, 3, ...
- **Terminating decimal** — A decimal that ends after a finite number of digits (e.g., 0.75, 3.5).
- **Repeating decimal** — A decimal in which a digit or group of digits repeats forever (e.g., 0.333..., 0.142857142857...).
- **Perfect square** — A whole number whose square root is also a whole number (e.g., 4, 9, 16, 25, 36, 49, 64, 81, 100).

---

## Concept Explanation

### What Makes a Number Rational?

A number is **rational** if it can be written as a fraction p/q where p and q are integers and q ≠ 0.

Every rational number has a decimal expansion that either **terminates** or **repeats**.

| Number | Fraction Form | Decimal | Type |
|--------|--------------|---------|------|
| 3/4 | 3/4 | 0.75 | Terminating |
| 1/3 | 1/3 | 0.333... | Repeating |
| −5 | −5/1 | −5.0 | Terminating |
| 2/7 | 2/7 | 0.285714285714... | Repeating |
| √4 | 2/1 | 2.0 | Terminating |

> A repeating block is written with a bar over it: 1/3 = 0.3̄, and 2/7 = 0.2̄8̄5̄7̄1̄4̄

### What Makes a Number Irrational?

A number is **irrational** if it **cannot** be written as a fraction of two integers. Its decimal expansion never ends AND never repeats — no pattern ever forms.

Famous irrational numbers:

| Number | Decimal (partial) | Why Irrational |
|--------|-------------------|----------------|
| √2 | 1.41421356237... | Not a perfect square |
| √3 | 1.73205080757... | Not a perfect square |
| π | 3.14159265358... | Proven non-repeating |
| e | 2.71828182845... | Proven non-repeating |

**Square roots of non-perfect squares are always irrational.**

So √2, √3, √5, √6, √7, √8, √10, ... are all irrational.
But √4 = 2, √9 = 3, √16 = 4 are rational because they simplify to whole numbers.

### The Quick Classification Test

Ask: *"Can I write this as a fraction a/b where a and b are integers?"*

- If **yes** → Rational
- If **no** → Irrational

### Why √2 Is Irrational (Informal Proof)

Suppose √2 = p/q in lowest terms (no common factors). Then:

2 = p²/q²  →  p² = 2q²

This means p² is even, so p must be even. Write p = 2k:

(2k)² = 2q²  →  4k² = 2q²  →  q² = 2k²

So q² is also even, meaning q is even too.
But if both p and q are even, the fraction p/q wasn't in lowest terms — **contradiction**.
Therefore, √2 cannot be written as any fraction. It is irrational.

### The Real Number System

All real numbers live on the number line. Rational and irrational numbers together make up the **real numbers**.

```
Real Numbers
├── Rational Numbers
│   ├── Integers  (..., −2, −1, 0, 1, 2, ...)
│   │   ├── Whole Numbers  (0, 1, 2, 3, ...)
│   │   └── Natural Numbers  (1, 2, 3, ...)
│   └── Non-integer fractions  (1/2, −3/4, 0.6̄, ...)
└── Irrational Numbers  (√2, π, √5, e, √7, ...)
```

## Worked Examples

### Example 1 — Classify each number

Classify as rational (R) or irrational (I): 0.625,  0.121212...,  √9,  √7

**Solution:**

- **0.625** → **Rational (R)**. It terminates and equals 5/8.
- **0.121212...** → **Rational (R)**. The block "12" repeats, so it equals 12/99 = 4/33.
- **√9 = 3** → **Rational (R)**. 9 is a perfect square; √9 = 3 = 3/1.
- **√7** → **Irrational (I)**. 7 is not a perfect square. Its decimal is 2.6457513..., with no repeating pattern.

---

### Example 2 — The π = 22/7 misconception

A student says "π = 22/7, so π is rational." Is this correct?

**Solution:**
No. The fraction 22/7 = 3.142857142857... (repeating). But π = 3.14159265358... (non-repeating). The fraction 22/7 is a close **approximation** of π, not exactly equal. Since π's decimal never repeats, π is irrational.

---

### Example 3 — Locating a square root

Between which two consecutive integers does √50 fall? Is √50 rational or irrational?

**Solution:**
- √49 = 7 and √64 = 8, so **√50 is between 7 and 8**.
- 50 is not a perfect square, so **√50 is irrational** (√50 ≈ 7.071...).

---

## Common Mistakes

**Mistake 1: Assuming all square roots are irrational.**
√4 = 2, √25 = 5, √100 = 10. These are whole numbers — perfectly rational. Always simplify square roots before classifying.

**Mistake 2: Confusing a long or wild-looking decimal with irrational.**
The decimal 0.142857142857... looks chaotic, but the block "142857" repeats. It equals 1/7 and is rational. The key is whether a pattern *eventually* repeats, not how long it takes.

**2.** A student writes 0.999... and claims it is less than 1. Is the student correct? Is 0.999... rational or irrational?

**3.** Which of the following are irrational?
√25,  √30,  √36,  √40,  √49

**4.** Sort into the correct category:

Numbers: 5/3,  −√2,  0,  √100,  π,  e (≈ 2.71828...),  0.8̄,  3.14

- **Rational:** ___________
- **Irrational:** ___________

**5.** True or False: The sum of two irrational numbers is always irrational. Give an example to support your answer.

**6.** Between which two consecutive integers does √85 fall? Is it rational or irrational?

---

## Answers

1. a) R  b) R (= 5/99)  c) R (= 4)  d) I  e) R  f) I
2. 0.999... = 1 exactly (it equals 9/9 = 1). The student is incorrect. It is rational.
3. √30 and √40 are irrational. √25 = 5, √36 = 6, √49 = 7 are rational.
4. Rational: 5/3, 0, √100, 0.8̄, 3.14 | Irrational: −√2, π, e
5. False. Example: √2 + (−√2) = 0, which is rational.
6. √81 = 9 and √100 = 10, so √85 is between 9 and 10. It is irrational (√85 ≈ 9.220...).
