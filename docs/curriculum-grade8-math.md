## Lesson Template

Every lesson follows this structure when written:

```
1. Learning Objective   — one sentence: what the student will be able to do
2. Key Vocabulary       — 3–5 terms with definitions
3. Concept Explanation  — clear prose with examples inline
4. Worked Examples      — 2–3 step-by-step solved problems
5. Common Mistakes      — 1–2 pitfalls students typically make
6. Practice Problems    — 4–6 problems with answers at the bottom
7. CA Standard          — e.g. "8.NS.1"
```
